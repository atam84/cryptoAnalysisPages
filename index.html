Frequently asked questions

Everything you need to know about the template and billing.

How do I publish this on GitHub Pages?
Create a new repository, add this index.html, then go to Settings → Pages and select the main branch. If you name the repo USERNAME.github.io, the site is live at that domain.
Can I change the colors and fonts?
Yes. Update the CSS variables under :root. You can also remove the Google Font link to use system fonts only.
Is there any JS framework?
No. This is dependency-free to keep things simple and fast.
How do I add analytics or a contact backend?
For analytics, paste your script before </body>. For contact, point the form action to a service like Formspree or your own endpoint.
